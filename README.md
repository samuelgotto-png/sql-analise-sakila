# Projeto SQL – Análise de Atores e Categorias (Sakila)

## Contexto
Este projeto utiliza o banco de dados Sakila, uma base de dados de exemplo amplamente utilizada para estudos e práticas em SQL.

## Objetivo
Analisar a relação entre atores, filmes e categorias, utilizando JOINs para cruzar informações entre múltiplas tabelas.

## Consulta Desenvolvida
- Relação entre atores e filmes
- Identificação das categorias dos filmes
- Uso de JOINs múltiplos e alias de tabelas

## Tecnologias Utilizadas
- SQL
- MySQL
- Banco de dados Sakila
