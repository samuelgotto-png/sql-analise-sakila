# Projeto SQL – Análise de Vendas por Funcionário (Sakila)

## Contexto
Este projeto utiliza o banco de dados **Sakila**, amplamente usado para estudos e práticas em SQL.

## Objetivo
Analisar o desempenho de vendas dos funcionários, identificando métricas importantes como total vendido, média de valores e número de vendas realizadas.

## Tabelas Utilizadas
- payment
- staff

## Métricas Analisadas
- Maior valor de venda
- Menor valor de venda
- Média dos valores
- Total de vendas
- Número de vendas por funcionário

## Técnicas Utilizadas
- JOIN entre tabelas
- Funções agregadas (MAX, MIN, AVG, SUM, COUNT)
- GROUP BY
- ORDER BY
- Filtros com WHERE

## Arquivos do Projeto
- `staff_sales_analysis.sql`: consulta SQL com análise de vendas por funcionário.
