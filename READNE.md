# Análise de Dados com SQL – Base Sakila

Este projeto tem como objetivo praticar análise de dados utilizando SQL,
a partir da base de dados Sakila.

## Objetivo
Realizar consultas SQL para analisar a relação entre atores, filmes e categorias,
utilizando JOINs, agregações e ordenações.

## Tecnologias utilizadas
- SQL
- MySQL
- MySQL Workbench

## Análises realizadas
- Listagem de atores
- Atores com maior número de filmes
- Quantidade de filmes por categoria
